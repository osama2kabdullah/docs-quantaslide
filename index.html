Installation - top

Installation is easy and quick.

We recommend you to read the Shopify Manual to have a clear idea about all aspects of theme customization.

Install Theme:
  1. Extract the zip file that id downloaded. Now log in to your Shopify store and go to "Online Store". Now click on Upload Theme and upload the "Quantaslide-1.0.0(upload me).zip" file. this file can be found in the extracted folder.
  2. By Default, your uploaded theme is unpublished. Another theme is running on your store. Now to publish the uploaded theme click "Publish" Button.
